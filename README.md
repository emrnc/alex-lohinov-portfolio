# Alex Lohinov - Product Designer

Personal portfolio website for Alex Lohinov, a Product Designer focused on clear, scalable digital products and thoughtful interface systems.

The site presents a compact profile, contact links, and selected product design work. It is built with Next.js and kept intentionally small so it can be easy to read, modify, and extend.

## Open Source

This portfolio is open source under the MIT License. Anyone can use it as a starting point for their own personal site, adapt the layout, replace the content, and build on top of it.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- CSS
- Lucide Icons

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run typecheck
npm run build
```
